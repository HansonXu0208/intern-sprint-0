## What is this?
A tiny static page that fetches the current Bitcoin price (USD) from CoinGecko.

## Files
- `index.html` — UI + fetch logic
- `test.js` — smoke test that calls CoinGecko and validates response shape
- `spec.md` — small implementation spec (scope, acceptance criteria, edge cases)

## Run locally
Option 1 (recommended): open directly
- Double-click `index.html` to open in your browser
- Click **Load price**

Option 2: serve via a simple static server (optional)
- Any static server works (not required for this sprint)
